# My Advanced Blog

A modern blog layout built using **HTML5, CSS3, and Bootstrap**.  
This project focuses on creating a clean and responsive **grid-based design** for displaying blog content .


# Features

-  Grid layout for organizing blog posts
- Clean and modern UI
- Fast and lightweight


# Technologies Used

- HTML5  
- CSS3  
- Bootstrap
